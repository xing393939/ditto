#### 单个用户删除接口

#### 接口地址
  * 正式接口：/userreview/user/delete

#### 请求参数说明
|  参数         |说明          |是否必选|
| ------------- |:-------------|:-----:|
| username      | 用户名 |必选    |

#### 返回示例
```javascript
{
    "err": 0,
    "msg": "success",
    "data": []
}
```

#### 返回结果说明
```javascript
2.1	 删除用户接口
```
